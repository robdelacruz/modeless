Modeless

search, view, edit notes


Keyboard Controls

Start typing to incrementally search notes.

ESC clears the search and starts over. (Vimium users: CTRL-Backquote also works.)

CTRL-L returns to search box text.

CTRL-X deletes the selected note.

Up/Down arrow keys to select from the matched notes.

ENTER to edit selected note, or create a new note.

Notes are saved automatically.

No data is sent to the server. All notes are stored locally in the browser's local storage.

Source: github.com/robdelacruz/modeless
